# 💳 Credit Card Fraud Detection Using Machine Learning

## 🚀 Project Overview
This project builds a machine learning system to detect fraudulent credit card transactions using classification algorithms. The dataset is highly imbalanced, making it a realistic financial fraud detection problem.

The project focuses on:
- Data preprocessing
- Handling class imbalance using SMOTE
- Model training and evaluation
- Explainable AI using SHAP

---

## 🎯 Problem Statement
Credit card fraud detection is critical for financial institutions. The challenge is that fraudulent transactions are extremely rare compared to normal transactions.

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Imbalanced-learn (SMOTE)
- Matplotlib, Seaborn
- SHAP (Explainable AI)

---

## 📊 Workflow
1. Data loading and exploration  
2. Feature scaling  
3. Handling imbalance (SMOTE)  
4. Model training (Logistic Regression, Random Forest)  
5. Model evaluation  
6. Explainability using SHAP  

---

## 📈 Results
- Random Forest performed best
- High ROC-AUC score achieved
- SMOTE improved recall for fraud class
- Model successfully identifies rare fraud cases

---

## 🧠 Explainability
SHAP was used to interpret model decisions and understand feature impact on fraud prediction.

---

## 📌 Key Insights
- Dataset is highly imbalanced
- Recall is more important than accuracy
- Tree-based models perform better
- Certain features strongly influence fraud detection
